FAQ

Common Questions

Do I need an account to use MaiPDF?

No. You can upload and share PDFs without creating an account. However, registering gives you a persistent control panel where you can manage all your links, replace files, and review access history at any time.

What can I do in the MaiPDF user control panel?

The control panel lets you view all your active PDF links, replace the file behind any link without changing the URL, update security settings (view limits, expiration, download permissions), check who opened your documents, and revoke access instantly.

Can I replace a PDF without sending a new link to recipients?

Yes. From the control panel you can swap the file behind any existing link. Recipients continue using the same URL, but they will see the updated document immediately.

Is the MaiPDF user account free?

Yes. Creating a MaiPDF account is free. Registered users get the full control panel, access history, and file management features at no cost.

What happens to links I created before I registered?

Links created without an account are not automatically attached to a profile. Going forward, any PDF you upload while logged in will appear in your control panel. You can still manage older links using their individual reading codes.
