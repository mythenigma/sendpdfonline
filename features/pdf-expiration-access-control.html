Expiration Settings

  • Time-based Expiration: Set specific dates for access to expire
  • View-based Expiration: Limit access after a certain number of views
  • Duration-based Expiration: Set access duration from first view
  • Automatic Expiration: Documents automatically become inaccessible
  • Custom Expiration Rules: Create specific expiration conditions
