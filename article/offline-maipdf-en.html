Offline PDF Sharing

Package PDFs into encrypted HTML files stored on USB drives or local networks. Each opening requires a 5ms online verification — files stay in your hands while you control access remotely.

Try Offline Package See How It Works

6 min read   March 23, 2026   Security

Uploading confidential PDFs to the cloud means surrendering control. Pure local encryption means no way to revoke access once the file is sent. MaiPDF's Offline Version bridges both worlds: your PDF is packaged into a self-contained encrypted HTML file that lives on your USB drive, NAS, or internal server — but every time someone opens it, a lightning-fast 5ms handshake confirms they still have permission. Files in your hands, access rules in the cloud.

Core Features

Eight Capabilities That Set Offline Mode Apart

Local HTML Packaging

Your PDF is encapsulated into a single HTML file. Copy it to a USB drive, private NAS, or air-gapped machine — no cloud upload required.

Online Verification on Open

A 5ms background handshake checks the read code, remaining views, and expiration date. No connection at open time means no access.

Read Code / Encode Dual Channel

Read Code grants view-only access. Encode lets you replace files, add views, or revoke permissions — a built-in "regret button" for every share.

Access Analytics

Every verification request logs IP address and timestamp. See who viewed, when access ended, and export the full audit trail to CSV.

No-Print / No-Copy

Browser-level disabling of print menus, right-click context menus, and keyboard shortcuts prevents unauthorized reproduction.

Dynamic Watermark

Automatically overlays date + IP address when the file is opened. Even screenshots can be traced back to the source.

HTTPS Verification

The authorization handshake uses an encrypted HTTPS channel, preventing man-in-the-middle attacks from tampering with permissions.

One-Time Reading Mode

Deducts a view on open and invalidates when the window closes. Prevents screenshots via background streaming and session locking.
