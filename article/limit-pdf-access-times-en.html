Diverse Access Limit Types

Single Access Limit

Documents can only be viewed once, becoming invalid immediately after viewing. Perfect for extremely sensitive confidential documents to ensure information isn't redistributed.

💡 Use cases: Temporary passwords, one-time notifications

Multiple Access Limits

Set specific access count limits like 3, 5, or 10 times. Documents automatically expire after exceeding the count, providing flexible distribution control.

💡 Use cases: Training materials, product introductions

User-Level Limits

Set different access count limits for different users. VIP users can view multiple times while regular users have restricted counts.

💡 Use cases: Member content, premium materials

Time + Count Dual Limits

Combine time and count restrictions, such as maximum 3 views within 24 hours, providing more granular control strategies.

💡 Use cases: Exam materials, temporary documents

Frequently Asked Questions

Q: Can I reset access counts after they're used up?
A: Yes. In the admin dashboard, you can reset access counts for documents that have reached their limit, or remove count restrictions entirely. Changes take effect immediately.
Q: Do multiple accesses by the same user count separately?
A: By default, each document opening counts as one access. However, you can configure IP or device-based deduplication so that the same IP or device only counts once within a certain timeframe.
Q: How can I check remaining access counts?
A: Administrators can view remaining access counts in real-time through the dashboard. For viewers, we can optionally display remaining counts on the document page.
Q: Do access limits affect search engine indexing?
A: Documents with access limits will not be indexed by search engines, which is also a way to protect document privacy. If you need SEO-friendly sharing, please use other sharing methods.
Q: Can I set different access counts for different people?
A: Yes. By creating different sharing links, you can set different access count limits for different user groups. For example, VIP users get 10 accesses, regular users get 3.
Q: Can access count limits be combined with other security measures?
A: Absolutely. Access count limits can be combined with password protection, expiration control, IP restrictions, email verification, and other security measures to form multi-layer protection.
Q: Can I immediately revoke all access permissions if security issues are discovered?
A: Yes. The admin dashboard has an emergency revocation feature that can immediately invalidate all sharing links and stop any further access. This is an important component of our security system.
