이메일 인증으로 안전한 PDF 접근

승인된 이메일 주소로만 문서 접근을 허용하여 보안을 강화하세요

MaiPDF의 이메일 인증 기능은 PDF 문서 공유 시 수신자의 이메일 주소를 검증하여 승인된 사용자만 문서를 접근할 수 있도록 합니다. 간단하면서도 효과적인 보안 방법으로 중요한 문서를 안전하게 보호할 수 있습니다.

이메일 인증 과정

문서 업로드

보호할 PDF 문서를 MaiPDF에 업로드합니다.

이메일 목록 입력

접근을 허용할 이메일 주소들을 입력합니다.

보안 링크 생성

이메일 인증이 필요한 보안 링크를 생성합니다.

이메일 검증

수신자가 이메일을 확인하고 인증을 완료합니다.

이메일 인증 기능

단일 이메일 인증

특정 이메일 주소 하나만 접근을 허용합니다.

다중 이메일 목록

여러 이메일 주소에 대한 접근을 동시에 허용합니다.

도메인 제한

특정 도메인(예: @company.com)의 이메일만 허용합니다.

시간 제한 인증

인증 후 일정 시간 동안만 접근을 허용합니다.

알림 기능

새로운 인증 요청이나 접근 시 알림을 받습니다.

접근 로그

누가 언제 인증하고 접근했는지 기록합니다.

이메일 인증의 이점

  • 간단한 보안: 이메일 주소만으로 효과적인 접근 제어
  • 사용자 식별: 정확한 사용자 신원 확인
  • 즉시 배포: 이메일로 바로 문서 공유 가능
  • 추적 가능: 문서 접근 및 공유 내역 추적
  • 추가 보안: 다른 보안 기능과 함께 사용 가능

사용 사례

기업 문서 공유

중요한 비즈니스 문서를 특정 직원이나 파트너에게만 공유할 때 사용합니다.

교육 자료 배포

등록된 학생들에게만 강의 자료나 과제를 제공할 때 유용합니다.

프로젝트 협업

프로젝트 팀원들에게만 관련 문서를 공유합니다.

법률 문서

클라이언트나 관련 당사자에게만 법률 문서를 제공합니다.

이메일 인증 설정 방법

  1. MaiPDF 플랫폼에 로그인
  2. 보호할 PDF 문서 업로드
  3. 보안 설정에서 '이메일 인증' 선택
  4. 허용할 이메일 주소 또는 도메인 입력
  5. 추가 보안 옵션 설정 (시간 제한 등)
  6. 보안 링크 생성 및 이메일로 공유
