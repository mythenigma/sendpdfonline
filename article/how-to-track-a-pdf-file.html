How to Track a PDF File

See who opened your PDF, when, and how often. Use built-in access records—no extra tools or sign-up.

Start Tracking Control Access

Article

Why Track PDF Opens?

Knowing when and how often a link was opened helps you follow up on proposals, confirm that investors saw updates, and spot unexpected sharing. MaiPDF keeps access records tied to your Read Code and Modification Code so you can query history anytime.

Steps

Track in 3 Steps

1

Share PDF

Upload, get the link, and save your Read Code (and Modification Code if you use it).

2

Open Access Records

Go to the Access Records page from the main site or your link result.

3

Query history

Paste Read Code (and Modification Code). Review who opened and when.

Tips

Getting the most from tracking

  • Keep your Read Code and Modification Code safe; you need them to view access records.
  • Combine tracking with view limits and expiry so links don’t live forever.
  • Use the same link when you replace the PDF—tracking continues on the same codes.
