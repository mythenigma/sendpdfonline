Control how many times a file can be opened. Great for proposals, contracts, and timed materials. Combine with expiry and watermark for stronger control.

Why Set View Limits?

A shared link can be opened again and again or forwarded. View limits cap how many times the PDF can be opened. Pair them with an expiration date and optional watermark so the link stays under your control.

Setup

How to Set View Limits

  1. Upload PDF.
  2. Set allowed open times.
  3. Set expiry date/time.
  4. Add watermark if needed.
  5. Share link and monitor access.

Best Practice

Recommended Combinations

Client proposal

10 views + 7-day expiry + watermark.

Internal memo

Unlimited views + no-download.

Sensitive brief

3 views + email verification.
