## Usage notes

- Use these as **concept/reference materials** for homepage hero, product-introduction posters, App DRM vs Online Sharing sections, and promotional videos.
- Core message: **Online Sharing = fast browser links/QR codes; App DRM = stronger app-based protection, screenshot-aware reader, device binding, revoke control, `.maipdf` files.**
- Keep trust claims honest: do not reuse fake customer logos or compliance badges unless verified.
- If using the ink/taiji style on public pages, adapt text and layout for real responsive HTML; do not paste unverified claims such as SOC 2 or specific customer names.
