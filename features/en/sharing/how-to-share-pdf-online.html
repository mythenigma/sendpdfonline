FAQ

Common questions about sharing

Is it free to share a PDF link?

Yes. You can generate a reading link quickly. Optional advanced controls may depend on the workflow you choose.

Can I replace the PDF but keep the same link?

Yes. Keep your modification code so you can update the file behind the link later.

Can I know if someone opened it?

Yes. You can use access logs/analytics, and enable Read Alerts via Telegram when available.

What’s the best way to share: email, chat, or QR?

Use chat for speed, email for formal delivery, and QR when offline distribution matters. A single link works everywhere.
